My goal was to leverage the powerful feature extraction capabilities of CNNs to accurately classify handwritten digit images from these datasets. Sentiment Analysis on Rate My Professor

I performed sentiment analysis on a dataset from RateMyProfessors.com, a website where students anonymously review university instructors. The dataset contained over 1.7 million reviews spanning 63,000 professors from 6,000 schools. I preprocessed the review text data using techniques like tokenization, stopword removal, and stemming. I then implemented and rigorously evaluated Bidirectional LSTM to classify each review 's sentiment as positive, negative, or neutral. For the LSTM, I employed word embeddings and parameter tuning. I achieved an F1 score of 0.93 using the Bidirectional LSTM model through extensive hyperparameter optimization over learning rate, batch size, and epochs. This project showcased my proficiency in building and optimizing natural language processing pipelines for sophisticated sentiment analysis tasks on real-world datasets with noisy user-generated text.
